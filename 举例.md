实验设置

模型：统一使用 Pythia-70M（HuggingFace 链接）

优化方式：无训练方法（不改变模型参数）

数据集：pg-19（超长文本，可取单个 sample 测试）、wikitext 等

评估指标：困惑度（PPL） + 加速效果

加速指标（至少包括）

TTFT：Time To First Token

TPOT：Time Per Output Token

Throughput：吞吐量

Total/average FLOPs：总/平均浮点运算次数
